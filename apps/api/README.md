# Splash-UI API

FastAPI backend skeleton for authentication, repository APIs, and workflow orchestration.

For local GitHub OAuth login, configure:

- `GITHUB_OAUTH_CLIENT_ID`
- `GITHUB_OAUTH_CLIENT_SECRET`
- `SPLASH_UI_SESSION_SECRET`
- `SPLASH_UI_API_BASE_URL`
- `SPLASH_UI_WEB_BASE_URL`

Use `http://localhost:8000` for the API callback base URL and `http://localhost:3000` for the web app in local development.
